\begin{frame}{Appendix: Price-study methodology notes}
  \footnotesize
  \begin{itemize}\setlength\itemsep{0.72em}
    \item The econometric study uses monthly observations from January 2012 to December 2024.
    \item It tests producer, wholesale, retail, and export prices in one integrated framework.
    \item Johansen cointegration identifies long-run relationships, while VECM estimates adjustment speed.
    \item Result interpretation emphasizes contractual pricing cycles over immediate shock pass-through.
    \item The export-price series is modeled separately due to different external market drivers.
  \end{itemize}
  \source{Research paper: Price relationships along the Czech beer value chain (ResearchGate, AgEcon Search).}
\end{frame}

    \item The forecast engine is coupled with a physical warehouse-flow redesign, not deployed in isolation.
    \item Governance changes include automated ordering logic and new ownership of replenishment decisions.

\begin{frame}{Appendix: Route-learning mechanics in last-mile delivery}
  \footnotesize
  \begin{itemize}\setlength\itemsep{0.72em}
    \item Static assumptions fail because unloading constraints vary by specific venue and street geometry.
    \item Machine-learning dispatch updates service-time priors from real stop behavior over time.
    \item If one pub repeatedly requires extra unloading time, future route plans absorb that local reality.
    \item Telematics feedback closes the loop between plan quality, delay patterns, and next-day routing.
    \item This is the operational mechanism behind route-density gains, not only a map-optimization feature.
  \end{itemize}
  \source{HOPI logistics materials, Descartes/FarEye route-optimization guidance, Czech transport case notes.}
\end{frame}

\begin{frame}{Appendix: Tank-beer digital stack detail}

  \begin{itemize}\setlength\itemsep{0.72em}

    \item Cellar operation enforces a narrow thermal control band to preserve quality at point of pour.
    \item Telemetry streams level, temperature, and flow data to central planning dashboards.
    \item Driver workflows use 2D code scanning on smartphones for high-confidence proof of delivery.
    \item Operational data is integrated into central planning systems used by dispatch and quality teams.
  \end{itemize}
  \source{Tank-beer technical references, Scandit case study, Prodac systems documentation.}

    \item Return-machine workflow runs image capture, barcode match, material check, and fraud filtering.

\begin{frame}{Appendix: Execution-risk and governance checklist}
  \footnotesize
  \begin{itemize}\setlength\itemsep{0.72em}
    \item Data-sharing agreements must define ownership and usage rights across producers, distributors, and retailers.
